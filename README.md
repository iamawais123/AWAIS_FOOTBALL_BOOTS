# AWAIS Football - Premium eCommerce Website

A modern, high-performance eCommerce website for AWAIS Football brand, built with Next.js, Tailwind CSS, and Framer Motion.

## 🚀 Features

### Core Features
- **Advanced Navigation Bar** - Sticky navbar with mega menu and smooth animations
- **Hero Section** - Full-screen cinematic hero with parallax effects
- **Featured Products** - Premium product grid with hover effects
- **Product Showcase** - Nike-style product presentation with rotating display
- **Collections** - Stylish collection cards with gradient overlays
- **Why Choose AWAIS** - Feature highlights with animated icons
- **Football Lifestyle** - Cinematic brand statement section
- **Testimonials** - Auto-sliding customer review carousel
- **Instagram Section** - Social media integration grid
- **Newsletter** - Email signup with discount offer
- **Premium Footer** - Multi-column footer with social links

### eCommerce Features
- **Shopping Cart** - Slide-in cart with quantity management
- **Wishlist** - Save favorite products
- **Search Modal** - Live product search with category filtering
- **Quick View** - Product preview modal with size selection
- **Product Filtering** - Filter by category, price, and more
- **Responsive Design** - Mobile-first, fully responsive layout

### Animations
- **Framer Motion** - Smooth page transitions and hover effects
- **Scroll Reveal** - Elements animate as you scroll
- **Hover Interactions** - Product cards lift and zoom on hover
- **Button Effects** - Ripple and scale animations
- **Parallax Scrolling** - Depth effects on scroll

## 🛠 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, GSAP, AOS
- **Icons**: Lucide React
- **Language**: TypeScript

## 📦 Installation

```bash
npm install
```

## 🏃 Running the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📱 Building for Production

```bash
npm run build
npm start
```
