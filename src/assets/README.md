# Assets Folder Structure

This folder contains all the images and media assets for the AWAIS Football website.

## Folder Structure:

- **products/** - Football boot product images
  - Organize by brand or category
  - Example: `nike-speed-pro-x.jpg`, `adidas-control-elite.jpg`
  
- **collections/** - Collection banner images
  - `speed-series.jpg`
  - `control-series.jpg`
  - `striker-series.jpg`
  - `street-football.jpg`
  
- **hero/** - Hero section images
  - `hero-main.jpg`
  - `hero-secondary.jpg`
  
- **lifestyle/** - Lifestyle and promotional images
  - `lifestyle-1.jpg`
  - `lifestyle-2.jpg`
  - etc.

## Image Guidelines:

- Recommended size: 800x800px for product images
- Recommended size: 1200x800px for collection banners
- Recommended size: 1920x1080px for hero images
- Use JPG or PNG format
- Keep file sizes optimized (under 500KB per image)

## Naming Convention:

Use lowercase with hyphens:
- `brand-model-name.jpg`
- `collection-name.jpg`
- `hero-section.jpg`
