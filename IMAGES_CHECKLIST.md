### ✅ Product Images (400+ images needed)
For each product, you need 2 images:
- Main image: `{brand}-{model}-{number}.jpg`
- Hover image: `{brand}-{model}-{number}-alt.jpg`

**Example for first few products:**
- [ ] `src/assets/products/awais-speed-pro-x-1.jpg`
- [ ] `src/assets/products/awais-speed-pro-x-1-alt.jpg`

## 🎯 Quick Start

1. Create the folder structure:
   ```
   src/assets/
   ├── hero/
   ├── collections/
   └── products/
   ```

2. Add your images following the naming convention above

3. The website will automatically use your local images!
